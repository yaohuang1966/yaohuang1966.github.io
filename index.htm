用 GitHub 輕鬆架站

GitHub 是全球最大的技術社群與軟體原始碼代管網站, 也是最受歡迎的開放原始碼專案協作平台, 雖然它本質上主要是 Git 版本控制的一個遠端儲存庫, 但同時也提供靜態網頁代管服務, 提供免費用戶 1GB 容量與每月 100GB 流量, 其雲端主機非常穩定, 很適合架設個人網站.
