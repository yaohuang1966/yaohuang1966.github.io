建立網站儲存庫

GitHub 的儲存庫相當於電腦硬碟根目錄下的子目錄, 即使是免費帳號也可以建立不限數量的儲存庫 (repository), 在這些儲存庫下又可以建立不限層數的子資料夾. 用 GitHub 架站時要建立一個特殊名稱的儲存庫, 此名稱格式為 : 帳號.github.io, 稱為網站儲存庫.

1. 新增儲存庫

登入 GitHub 官網 https://github.com 在首頁按 "Create repository" 鈕, 或右上角按 "+" 選 "new repository" :

2. 設定儲存庫 :

在 "Repository name" 欄填入特定的儲存庫名稱, 格式為 : 帳號.github.io, 例如 GitHub 帳號是 yaohuang1966, 則必須填入 yaohuang1966.github.io, 勾選 "Public" 與 "Add a README file" :




最後按底下的 "Create repository" 即建立了網站儲存庫 :

事實上, 建立一般的儲存庫方式與上面一樣, 只是 Repository name 欄位可隨心所欲命名而已. 其次, 微軟在 2018 年併購 GitHub 後允許免費帳戶可建立私有儲存庫 , 故若所建立之儲存庫不想開放公眾存取, 請在設定時勾選 Private.
