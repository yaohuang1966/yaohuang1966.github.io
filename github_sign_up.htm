註冊 GitHub 帳號

使用 GitHub 架站之前必須先註冊一個 GitHub 帳號, GitHub 提供免費與付費帳號, 對一般使用者來說, 免費帳戶的資源即足夠使用 (1GB 容量 + 100GB 每月流量).

連線 GitHub 官網

首先連線 GitHub 官網首頁 https://github.com , 然後按 'Sign up' 鈕進入註冊頁面 :

輸入 Email :

輸入可收信的郵件信箱 (例如 Gmail), 稍後要收取驗證碼之用 :

設定密碼 :

密碼至少 8 個字元, 必須包含一個小寫字母與一個數字 :

設定帳號 :

GitHub 帳號是獨一無二的, 也是網址的開頭, 例如帳號是 abc 則個人網站的網址就是 https://abc.github.io, 而且登入 GitHub 時可用帳號或 Email 登入 :
